# 💰 Decentralized Tip Jar

A simple and elegant smart contract that allows anyone to send STX tips to the contract owner. Perfect for content creators, developers, or anyone who wants to receive decentralized tips! 🚀

## ✨ Features

- 💸 **Send Tips**: Anyone can send STX tips to the contract owner
- 📊 **Tip Tracking**: Complete history of all tips with sender information
- 💬 **Tip with Messages**: Send tips along with custom messages
- 🔄 **Bulk Tipping**: Send tips to multiple recipients at once
- 📈 **Statistics**: View comprehensive tip statistics and analytics
- 🛡️ **Owner Controls**: Contract owner can withdraw tips and toggle contract status
- 🏆 **Tipper Rankings**: Track generous tippers and calculate averages

## 🚀 Quick Start

### Prerequisites
- [Clarinet](https://github.com/hirosystems/clarinet) installed
- Basic understanding of Stacks blockchain
